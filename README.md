# Projeto Transporte — Estrutura de Pilha (C# Console)

Este projeto implementa o controle de veículos em garagens utilizando **pilhas (Stack)**, conforme atividade acadêmica proposta.

## Funcionalidades

- Cadastro de veículos
- Cadastro de garagens
- Início e encerramento de jornada
- Registro de viagens entre garagens
- Relatórios de veículos e viagens
- Controle por pilha (último veículo que entra é o primeiro que sai)
